## 동기

나는 React, Vue를 사용하면서 어느 정도 웹페이지를 만들 수 있다고  생각했습니다. 문득 웹페이지를 만들면서 나는 library와 framwork만 조금 사용할 줄 아는 것 아닐까  생각을 하게 되었다. 그래서 Vanilla js만으로 웹페이지를 만들어 보겠다는 결심을 하게 되어 제작하게 되었습니다.
